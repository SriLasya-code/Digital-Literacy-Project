##  Project Overview
This project is part of the CSE0001 Digital Literacy course.The aim of this project is to understand and apply essential digital skills such as building an online presence, using collaborative tools, maintaining professional communication, and staying safe online.

The project is divided into five tasks, each focusing on a different aspect of digital literacy.

---

##  Project Structure

- `task-1-presentation/`  
  Digital Literacy Awareness Infographic (Canva design)

- `task-2-portfolio/`  
  Screenshots of GitHub, LinkedIn, and other profiles

- `task-3-platforms/`  
  Coding platform activity + Google Form and responses

- `task-4-email-etiquette/`  
  Professional email drafts + social media checklist

- `task-5-cybercrime/`  
  Case study + prevention checklist

- `report/`  
  Final Project Report
